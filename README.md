# Marketing Campaign Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a marketing campaign dataset to identify customer behavior patterns, campaign effectiveness, and opportunities to improve ROI.

## Objectives
- Analyze customer demographics and spending behavior.
- Evaluate campaign acceptance rates.
- Identify high-value customer segments.
- Generate business recommendations to improve marketing ROI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Wine products generated the highest customer spending.
- High-income customers contributed more revenue.
- Campaign acceptance rates indicate opportunities for better targeting.
- Personalized promotions can improve conversion rates.
